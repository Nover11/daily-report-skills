# Daily Report Skills

This repository contains Agent Skills for generating daily work reports.

## Skills

### intern-daily-report

Generate a copy-paste-ready intern daily report from work conversations with an Agent.

## Install

In a Skill-compatible Agent such as Claude Code, Codex, OpenClaw, or similar, ask:

```text
帮我安装这个 skill：https://github.com/<your-username>/daily-report-skills/tree/main/intern-daily-report
```

Replace `<your-username>` with your GitHub username or organization.

## Usage

```text
生成今天的实习生日报
```

```text
预览今天会写进日报的事项
```

```text
这个不要写进日报
```

```text
这件事计入今天日报
```
